# Privacy & ZK Blockchains · Landscape | 隐私与 ZK 区块链 · 全景图

A bilingual (EN ↔ 中文) curated map of 28 blockchain projects building cryptographic privacy or zero-knowledge verification, organized into 5 categories by technical approach.

## Categories

1. **UTXO Shielded-Pool L1s** — Zcash, Monero, Aleo, Iron Fish, Penumbra, Namada
2. **TEE-Based Confidential Chains** — Secret Network, Oasis Sapphire, Phala Network
3. **General-Purpose ZK Rollups** — zkSync Era, StarkNet, Polygon zkEVM, Scroll, Linea, Polygon Miden, Taiko
4. **Privacy on EVM (Smart-Contract Layer)** — Aztec Network, Railgun, Nocturne Labs
5. **Sovereign & Specialized ZK Chains** — Psy Protocol (featured), Mina, Manta Network, Anoma, Aleph Zero

## Disclaimer

Curated, not comprehensive. Categories overlap. Inclusion is not endorsement; exclusion is not criticism. Statuses as of April 2026. Bridge-only privacy mixers and pre-mainnet zkVMs are deliberately omitted — the focus is settlement chains.
